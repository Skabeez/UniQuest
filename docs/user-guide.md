# UniQuest — User Guide

Welcome to UniQuest! This guide helps you quickly get familiar with the app and its features.

## What You Can Do
- Create an account and sign in via email
- Explore the campus map with tappable pins and info cards
- Complete onboarding walkthrough and tutorials
- View tasks, missions, achievements, and profile settings
- Navigate with smooth transitions and a dark theme optimized for mobile

## First Run
1. Open the app and tap "Continue to Home".
2. If asked, complete onboarding to set up your preferences.
3. Use the bottom navigation to move between Home, Map, Profile, and Tasks.

## Campus Map
- Tap location pins to see the name, photo, and description.
- If an image fails to load, the app shows a helpful fallback image.
- Pin examples: Library (UL), CAS, CCJ, CED, CAFENR, CTHM, ICON, U‑Mall, Chapel.

## Account & Profile
- Sign up with email and password.
- After sign up, check your inbox for a confirmation email.
- Update profile details and cosmetics in Settings.

## Tasks & Missions
- Active Missions: track progress on current goals.
- Todo List: manage quick tasks with priority and status badges.
- Achievements: see badges earned as you participate.

## Notifications
- Enable notifications when prompted to receive important updates.

## Troubleshooting
- Stuck on a white screen? Try a restart; the app uses dark backgrounds to avoid white flashes during asset loads.
- Fonts or images missing? Ensure you have an internet connection (for remote resources) and report any asset problems to support.

## Support
If you run into issues, open a ticket or report a bug via the repository Issues tab.
